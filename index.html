Мы всегда стараемся быть надежным партнером для крупных оптовых покупателей и единичных заказчиков специфических эксклюзивных отливок из стали, отливок из бронзы, отливок из чугуна, а также - оснастки для литья. Детали из чугуна и детали из стали к промышленному оборудованию предлагаются нашим предприятием в широком ассортименте. Модельная оснастка
Качественно изготовленная модельная оснастка - это залог успеха при производстве литейной продукции. Формы для литья, требуют особого внимания, так как, в случае ошибки, ущерб понесет производитель. Потому, высококвалифицированные сотрудники, это не роскошь, а реальная необходимость и фактически, основа успешного производства. Опытные сотрудники и современное оборудование - присутствующий фактор, нашего модельного производства. Для производства модельной оснастки, используются следующие станки: специальные станки с ЧПУ, либо фрезерные или токарные станки. ЧПУ (числовое программное управление) - заботится о качестве изготовления модельной оснастки, устраняя возможные ошибки, которые можно сослать на "человеческий фактор". Мы специализируемся на производстве литейной (модельной) оснастки. Модельная оснастка для всех видов металлов: • для дробильно-размольного оборудования (плиты, конуса, шнеки, шары, щеки); • для подъемно-транспортных механизмов (ролики, крюки, звездочки, подвески, направляющие); • для приводных механизмов (зубчатые колеса, шестерни, колеса электродвигателей, насосов и редукторов/мультипликаторов, корпуса).
